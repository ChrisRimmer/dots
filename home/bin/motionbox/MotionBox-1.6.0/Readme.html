MotionBox is a Video Browser for Motion Freedom.

Built to access and browse decentralized video sources.

Built to load and play multiple video resources.

My name is Benjamin Arnaud aka bunjee.

I'm building MotionBox to empower people:

Let's build a Video Browser together.

Changelog

  • 1.6.0: Universal backends, macOS support, subtitles implementation.

  • 1.5.0: Light themes, 64 bit, playback history.

  • 1.4.0: Torrent search, Qt 5, VLC 3.0.

  • 1.3.0: Torrent streaming, dynamic covers, improved full screen.

  • 1.2.0: Complete audio support, playback speed, native window.

  • 1.1.2: Improved interface, fast startup, wide search bars.

  • 1.1.1: Buttons highlight, cleaner interface, components style.

  • 1.1.0: Complete track browser, dynamic loading, video files.

  • 1.0.1: Window and player inputs, url parsing, contextual menus.

  • 1.0.0: Initial release.

Technology

MotionBox is built in C++ with Sky kit.

Sky is built on the Qt framework.

The GUI uses QML and the player uses VLC and libtorrent.
